Conectar con Google Sheets — un solo paso

Seguí estos pasos para generar tu URL de conexión (gratis, tarda 3 minutos, se hace una sola vez):

  1. Abrí tu Google Sheet → menú Extensiones → Apps Script
  2. Borrá todo el contenido y pegá el código de abajo → Guardar (ícono del diskette)
  3. Click en Implementar → Nueva implementación → Tipo: Aplicación web → Ejecutar como: Yo → Quién puede acceder: Cualquier usuario → Implementar
  4. Copiá la URL que aparece y pegala acá abajo

Código para Apps Script:

function doGet() { var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName("Chispa Tareas") || ss.getSheets()[1]; var data = sheet.getDataRange().getValues(); return ContentService .createTextOutput(JSON.stringify(data)) .setMimeType(ContentService.MimeType.JSON); }

⚠️ En el paso 3, la primera vez te va a pedir que autorices el acceso — aceptá con tu cuenta de Google.
